# WatchGPT
A ChatGPT client for Android watch

## Features
1. Chat with AI
2. Save system prompts and apikey
3. Switch models
4. Markdown support

## Goals
1. Chat history
2. Code highlights
3. UI optimize
4. System Prompts library
5. Multi languages support
